Levels completed: 
Level 1 –
Steps:

Abstract Class

Create Animal (abstract) with at least one abstract method (e.g., makeSound() or move()).

Subclasses

Create at least 3 concrete subclasses (e.g., Bird, Fish, Mammal).

Each must:

Override the abstract method(s)

Have a constructor to set a variable (e.g., name or species)

Implement unique sound and movement behaviors

Interface

Create 1 interface (e.g., Flyable, Swimmable, Runnable) with at least one method.

Implement it in the appropriate subclass(es).

OOP Principles

Use private variables with public getters/setters

Use extends and implements correctly

Apply @Override on overridden methods

Main.java

Create and store all animals in an ArrayList<Animal>

Loop through and call their methods (polymorphism)

Output each animal’s name, movement, and sound

Level 2 – Skill Expansion 

Steps:

Create a second interface (e.g., Walkable, Climbable, Burrower) with at least one method.

At least 2 animals must implement both interfaces.

Add new methods/behaviors for the second interface.

Update Main.java to demonstrate these new methods.

Level 3 – Professional Quality 

Steps:

Refactor into packages (e.g., animals, interfaces, main).

Create ZooManager that:

Loads animals into a collection

Displays a menu (e.g., “List All,” “Make All Move,” etc.)

Uses Scanner for user interaction

Add JavaDoc for all public classes and methods.

Add exception handling for invalid input.
